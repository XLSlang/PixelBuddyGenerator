# 🧬 Pixel Buddy - 为所有人服务的本地AI生成工具

> *因为 AI 不应该成为少数人的特权*

## 📖 项目缘起

近年来，AI 生成视频服务逐渐走向云端和付费，每次生成都要上传数据、等待队列、支付费用。这让我不禁思考：

**一个真正的创作者工具，为什么要依赖网络？为什么要按次收费？为什么要上传自己的数据？**

所以我找回了编程路上最可靠的老朋友 —— **C 语言**。

当前版本还是一个 **2D 像素角色生成器**，但这只是起点。我希望它能成为一个：
- 🆓 **完全免费**，无任何隐藏费用
- 🔒 **本地运行**，你的数据永远留在你的电脑上
- 🌍 **开源协作**，由社区共同打磨
- 🚀 **持续演进**，从像素到动画，从角色到场景

## 🎯 为什么是 C 语言？

| 对比项 | 云端方案 | Python方案 | **C语言方案** |
|--------|---------|-----------|--------------|
| 运行成本 | 按次/按月收费 | 需安装运行时 | **零依赖，直接运行** |
| 隐私安全 | 数据需上传 | 本地但易反编译 | **原生执行，难以篡改** |
| 跨平台 | 需浏览器 | 需解释器 | **单 exe 随处运行** |
| 性能 | 受网络限制 | 中等 | **极致轻量** |

## 🔧 当前功能

- ✅ 自然语言描述 → 角色DNA解析（内置轻量AI）
- ✅ 多种风格模板（战士、法师、骑士等）
- ✅ 高清导出（最高4K BMP）
- ✅ 3种渲染算法（像素风/平滑/复古）
- ✅ 纯命令行，无任何网络请求

## 🚀 编译与运行

```bash
